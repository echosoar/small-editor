Small Editor Manual Test

Test Instructions

1. Multi-line Tab Indentation

• Type some lines of text

• Select multiple lines

• Press Tab to indent all selected lines

• Press Shift+Tab to dedent all selected lines

2. HTML Tag Auto-completion

• Type "<div" and then press ">"

• Should auto-complete to "<div></div>" with cursor positioned between tags

• Try with other tags like "<span", "<p", etc.

3. Floating Point Precision Fix

• Type "0.1 + 0.2 =" and press "="

• Should show "0.3" instead of "0.30000000000000004"

• Try other calculations like "0.1 * 3 ="

4. Image Drag and Drop

• Drag an image file into the editor

• Should show "![filename](uploading...id)" temporarily

• Will show "![filename](upload-failed)" since no upload handler is configured
